# Telegram Nexus

---

**NEXUS COMMAND** — это мощная панель управления для автоматизации Telegram-аккаунтов. Позволяет управлять неограниченным количеством сессий через веб-интерфейс, выполнять массовую рассылку и быстро добавлять новые аккаунты

---
## Установка и первый запуск
```bash
git clone https://github.com/Ghostoraner/Telegram-Nexus.git
cd Telegram-Nexus
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn main:app --reload
```
Программа будет ждать вас тут >  http://127.0.0.1:8000
